# Sage

Sage is an executive productivity assistant built for high-signal daily execution.

It helps a leader move through a practical workflow:
- see today's priorities
- prepare for critical meetings
- capture and debrief notes
- turn follow-through into tasks
- review weekly load and decision risk

## Product

- `Today`: executive command center
- `Week Planner`: weekly load and routing into action
- `Meeting Brief`: pre-meeting context and decision prep
- `Debrief`: post-meeting decisions, actions, and follow-through
- `Tasks`: execution queue and prioritization
- `Insights`: executive operating signals
